# Business Inteligence Assignment 3

This is the repository for the 3rd assignment for course Business Inteligence.

## Dataset
We will create a classification model using the Spotify 1 Million Tracks dataset (https://www.kaggle.com/datasets/amitanshjoshi/spotify-1million-tracks). The songs will be classified in 5 categories based on their popularity score:
- Very bad (0-20)
- Bad (21-40)
- Average (41-60)
- Good  (61-80)
- Very good (81-100)
